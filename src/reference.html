Референс-лист

01.

Крупнейший проект по обустройству шельфового месторождения в Каспийском море

клиент: лукойл

|

2019

Завод РУСТ-95 принимал участие в реализации крупнейшего проекта Лукойл по обустройству шельфового месторождения им. В.И. Грайфера в Каспийском море.

Осуществлена поставка запорной и регулирующей арматуры (клапаны запорные и регулирующие, регуляторы давления, краны шаровые)  для верхнего строения ледостойкой стационарной платформы. Арматура выполнена в специальном морском исполнении по индивидуальным требованиям Заказчика. Краны шаровые, предназначенные для системы ПАЗ изготовлены в огнестойком исполнении с системой сброса давления на седле запорного органа.
